An AI-powered tool that screens resumes against job descriptions and provides a skill match evaluation.

## Objective
To automate the resume evaluation process using NLP and ML, reducing manual effort in candidate shortlisting.

---

## Features

| Feature | Description |
|--------|-------------|
| Resume Upload (PDF) | Upload resume in PDF format |
| Skill Extraction | Extract relevant skills from the resume |
| JD Comparison | Compare extracted skills with job description |
| Skill Match Score | Percentage match based on similarity |
| Hiring Decision | Accept / Hold / Reject recommendation |
| Streamlit UI | Fast and smooth user interface |

---

## Technologies Used

| Technology | Purpose |
|-----------|---------|
| Python | Logic development |
| Streamlit | Web UI |
| pdfplumber | Extract resume text |
| Scikit-Learn | Skill similarity scoring |

---

## Workflow

1. Upload resume in PDF format.
2. Extract skills and keywords from the resume.
3. Compare extracted skills with the job description.
4. Calculate the match score.
5. Display strengths, missing skills, and the final recommendation.
